👜 Women’s Handbag Store & Warehouse Management System

📌 Overview

This project implements a database system to manage a women’s handbag store, including products, sales, purchases, and warehouse operations.

⚙️ Features

Manage products, categories, and inventory

Track customer orders and sales

Handle supplier purchases and restocking

Monitor warehouse stock and transfers

Generate reports for sales and inventory

🧪 Functionality

Stores and organizes data using relational tables

Uses queries to track inventory, orders, and suppliers

Supports efficient management of store and warehouse operations

🛠️ Technologies Used

MySQL

Python

Tkinter

Database Design (ERD, SQL)

▶️ How to Run

Set up the database in MySQL

Run the Python application

Use the interface to manage store operations

🎯 Key Concepts

Database design (ERD & normalization)

SQL queries and relationships

Inventory management systems
GUI development with Tkinter
